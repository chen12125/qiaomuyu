# 商城系统设计文档

## 商城概述

用户可以使用积累的功德在商城中兑换各种物品，包括外观、音效、主题等。

## 购买流程

1. 用户进入商城界面
2. 浏览商品列表
3. 选择商品查看详情
4. 确认购买（扣除相应功德）
5. 立即生效使用
