# 敲木鱼安卓应用

一个有趣的敲木鱼安卓应用，结合功德系统和商城功能，让敲木鱼更有成就感。

## 功能规划

### 核心功能
- [ ] 木鱼敲击音效
- [ ] 触摸屏幕敲击木鱼
- [ ] 敲击计数统计
- [ ] 敲击频率统计
- [ ] 简单的动画效果

### 功德系统
- [ ] 点击木鱼获得功德（1点击=1功德）
- [ ] 功德等级系统
- [ ] 功德排行榜
- [ ] 成就系统

### 商城功能
- [ ] 功德商店
- [ ] 木鱼外观兑换（金木鱼、玉木鱼等）
- [ ] 音效包兑换
- [ ] 背景主题兑换
- [ ] 特效动画兑换

### 其他功能
- [ ] 设置界面（音量调节、音效选择）
- [ ] 每日签到
- [ ] 分享功能

## 技术栈

- Android Studio
- Kotlin/Java
- Android SDK
